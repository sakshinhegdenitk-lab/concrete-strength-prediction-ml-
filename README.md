# Concrete Strength Prediction Using Machine Learning

This project uses Machine Learning to predict the compressive strength of concrete based on material and mix-related parameters.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Overview
The project applies data preprocessing and regression techniques to analyse concrete data and predict compressive strength.

## Model Evaluation
The model is evaluated using regression performance metrics to understand prediction accuracy.
